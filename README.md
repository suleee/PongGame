# Pong Game

A basic pong game using SVGs.

![alt tag]()


## Setup

**Install:**

`> npm i`

**Run:**

`> npm start`


## Keys

**Player 1:**
* a: up
* z: down
* s: fire shot

**Player 2:**
* ▲ : up
* ▼: down

**Share code:**
* enter: start game
* space bar: pause the game
* refresh: re-start the game
